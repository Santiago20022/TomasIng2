# 02 - Espina de pescado

Problema central: **fragmentacion del ecosistema de voluntariado en Colombia**.

```
              Procesos                Tecnologia                     Organizaciones
                |                         |                                |
  Procesos     |    Falta de plataforma   |     Pocas org tienen           |
  manuales      |    centralizada          |     equipo digital propio     |
  de difusion   |                         |                                |
  ------------>+-------------------------+--------------------------------+----> Fragmentacion
                |                         |                                |
                |                         |                                |
            Personas               Datos                              Politicas
              |                       |                                  |
   Brecha digital           Sin indicadores          Iniciativas dispersas
   por estrato              de impacto reales        sin estandar comun
```

| Categoria | Causas principales |
|---|---|
| Procesos | Difusion manual, inscripciones por whatsapp/forms ad-hoc. |
| Tecnologia | Ausencia de un portal unico, brecha digital diferenciada por estrato. |
| Organizaciones | Pequenas org no tienen equipo digital ni presupuesto para mantenerlo. |
| Personas | Falta de tiempo, desconocimiento de oportunidades cercanas. |
| Datos | Inexistencia de indicadores de impacto compartidos. |
| Politicas | Iniciativas dispersas, sin un estandar nacional. |
