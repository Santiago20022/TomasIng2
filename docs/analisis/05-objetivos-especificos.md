# 05 - Objetivos especificos

1. **Analizar** las necesidades y barreras tecnologicas de voluntarios y organizaciones sociales en Colombia mediante tecnicas de elicitacion para definir requisitos inclusivos.
2. **Disenar** una arquitectura de software basada en modelado UML y tecnica "UML en Color" que garantice escalabilidad y experiencia simplificada para todos los estratos.
3. **Implementar** un modulo de inscripcion y aprobacion que conecte de manera eficiente los intereses del voluntario con las vacantes presenciales disponibles.
4. **Desarrollar** una herramienta de analitica social que permita a organizaciones y administradores generar reportes automaticos de impacto, eliminando seguimiento manual.
5. **Validar** la coherencia tecnica y la trazabilidad mediante matrices de trazabilidad globales, asegurando que cada RF responde a una necesidad real del usuario.
