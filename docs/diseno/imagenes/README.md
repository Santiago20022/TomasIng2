# Imagenes de diseno

En esta carpeta deben quedar las capturas/exports de:

- Wireframes de baja fidelidad (E16) — uno por pantalla del mapa de navegacion.
- Capturas del prototipo Figma (E17).
- Diagramas exportados desde la herramienta UML usada.

Mientras tanto, las descripciones textuales de cada artefacto estan en los archivos `.md` correspondientes en `../`.
