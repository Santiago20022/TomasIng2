# E1 - Tabla de contexto

| Actor externo | Entrada al sistema | Salida del sistema |
|---|---|---|
| Voluntario | Datos de registro, filtros de busqueda, solicitud de inscripcion, reseñas, mensajes | Catalogo de actividades, confirmaciones, notificaciones, conversaciones |
| Organizacion | Datos de registro, formulario de actividad, decisiones sobre inscripciones, mensajes | Listado de actividades publicadas, lista de inscripciones, notificaciones |
| Administrador | Filtros de organizaciones, cambios de estado de verificacion, peticion de reporte IA | Listado global de organizaciones e inscripciones, estadisticas, informe IA |
| Servicio Groq (IA) | Prompt construido con estadisticas | Texto del informe ejecutivo |
